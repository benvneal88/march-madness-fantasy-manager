# Changelog

All notable changes to this project are documented in this file.

## v0.1.1 - 2026-03-19

### UI
- Updated Fantasy Teams player row layout to inline status formatting:
  - Player Name | Active/Eliminated | School Name | Pick #
- Refreshed Fantasy Teams status styling:
  - Active uses a light blue/gray treatment.
  - Eliminated uses a light red treatment.
- Added an "Only Available" checkbox filter on Draft Night Team Rosters to hide drafted and injured players.
- Added an admin draft setting to show/hide the Play-In (`P`) round in fantasy manager tables.
